# SkinBloom

A Python-based skincare assistant that generates personalized skincare routines based on skin type and concerns.

## Features
- Personalized routine generator based on skin type/concerns
- Ingredient explorer
- Save and revisit your skincare profile
- Built across 5 modular files (main, routines, ingredients, tips, basics) instead of one script

## How to run
1. Clone the repo: `git clone https://github.com/samihasoyshee/SkinBloom.git`
2. Run: `python main.py`

## What I learned
Structuring a Python program across multiple files/modules instead of a single script.
